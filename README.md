# 📊 Netflix Data Analysis & Visualization

A beginner-friendly data analysis project built with **Python**, **Pandas**, and **Matplotlib**. This project explores the Netflix Titles dataset by cleaning the data and creating various visualizations to uncover trends and insights.

## 📌 Features

* Data cleaning using Pandas
* Movies vs TV Shows comparison
* Content rating distribution
* Movie duration analysis
* Release year trends
* Top 10 countries by number of titles
* Movies and TV Shows comparison over the years

## 🛠️ Tech Stack

* Python
* Pandas
* Matplotlib

## 🚀 How to Run

1. Clone the repository.
2. Install the required libraries:

```bash
pip install pandas matplotlib
```

3. Navigate to the project folder:

```bash
cd "Netflix Project"
```

4. Run the project:

```bash
python main.py
```
